# autored_project
프로젝트 개요

본 프로젝트는 OpenCV 라이브러리를 활용한 차선 인식 및 추종 알고리즘을 구현한 지능형 라인 트레이서 미니카입니다.

Hough 변환을 이용해 차선을 감지하고, 차선의 위치와 형태에 따라 조향과 모터를 제어하여 안정적이고 부드러운 자율 주행을 수행합니다.

하드웨어 제어는 gpiozero 라이브러리를 사용하여 Raspberry Pi의 GPIO 핀을 통해 조향과 모터를 정밀하게 제어합니다.

차선 인식:

OpenCV를 사용한 실시간 영상 처리.
    
ROI(Region of Interest) 설정을 통한 처리 영역 최적화.
    
Canny Edge Detection 및 HoughLinesP를 이용한 차선 추출.
    
P-제어:

화면 중앙과 차선 간의 거리를 서브모터 입력 범위(-1~1)로 정규화한 위치 오차값인 RX와 LX를 구하고 이 값이 클수록 차선이 한쪽으로 쏠렸다는 뜻이 되서 값의 부호를 반대로 뒤집고 가중치를 곱해 차선 반대방향으러ㅗ 핸들을 꺾어 중심을 잡도록 제어합니다. 
    
주행 안정성:

직진 보정 오프셋: 차량의 기계적 쏠림을 보정하여 직진 안정성 확보.
    
EMA 스무딩: 조향값에 지수 이동 평균을 적용하여 급격한 조향 변경을 억제하고 주행을 부드럽게 합니다.
    
강제 직진 모드: 양쪽 차선이 모두 감지될 경우 P-제어를 0으로 설정하여 직진 안정성을 극대화.
    
곡선 주행 최적화: 왼쪽/오른쪽 차선만을 추종할 때 각각 다른 P-제어 이득을 적용하여 곡선 주행 성능 최적화.

제어 알고리즘: 차선 중심점과의 오차를 기반으로 하는 P-제어를 주축으로 사용합니다.
